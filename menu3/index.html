WEB SCRAPING

AUGUST 22, 2023

In this project, we scrape data from the beer section of the [Boozy](https://www.boozy.ph) website using the Beautiful Soup library in Python. Specifically, for each product in the said section, we extract the following: 1. Name 2. Price 3. Star Rating 4. Number of Reviews A screenshot of the first page in the beer section of the website is shown below.

Baguio Craft Brewery

The program for data scraping is shown in the Python file named ['scraper'](). In this implementation, we construct a Pandas dataframe where names are stored as string, prices and ratings as floating point numbers, and number of reviews as integers.
